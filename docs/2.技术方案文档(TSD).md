# Trustavo 理财方案规划工具 - 技术方案文档

## 1. 技术栈选型
### 1.1 前端技术栈
- 核心框架: React 18 + TypeScript 5
- UI框架: Ant Design 5
- 状态管理: Zustand
- 样式方案: TailwindCSS
- 构建工具: Vite
- 开发工具: ESLint + Prettier

### 1.2 图表解决方案
- 图表库: @ant-design/charts
- 数据可视化: Line Charts

### 1.3 工具库
- 格式化: 自定义format工具
- PDF生成: html2pdf.js
