## 2. 评分算法
### 2.1 产品评分因素
- 收益率权重: 40%
- 灵活性权重: 30%
- 风险等级权重: 20%
- 品牌因素权重: 10%

### 2.2 评分计算
```typescript
function calculateProductScore(product: Product): number {
    const weights = {
        returns: 0.4,
        flexibility: 0.3,
        risk: 0.2,
        brand: 0.1
    };
    
    return (
        product.returns * weights.returns +
        product.flexibility * weights.flexibility +
        product.risk * weights.risk +
        product.brand * weights.brand
    );
}
```
